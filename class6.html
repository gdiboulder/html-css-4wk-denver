Background-image

Background Image

We can add background images to divs just like we do background colors.

#header {
background: url(../images/cacti.jpg) no-repeat top left;
color: #ffffff;
text-align: center;
}

Background-size

We can adjust the size of the image by using a property called: background-size 

#header {
background: url(../images/cacti.jpg) no-repeat top left;
color: #fafafa;
text-align: center;
height: 350px;
padding: 25px 0;
background-size: cover;
}

We can apply a value of "cover" so that we can scale the image to the largest size such that both its width and its height can fit inside the content area.

RGBA & Opacity

rgba colors & opacity

In CSS, we can choose colors with hexadecimal #0000FF or rgb(0, 0, 255)

In CSS3, there are ways to control opacity of a color too!

  • rgba - controls the opacity of a color
  • opacity - controls the opacity of an element
