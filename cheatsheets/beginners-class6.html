Intermediate CSS - The fun parts!

Background Image

  • Apply a background image to a div or section.

Most Elements:


header {
    background-image: url('PATH/TO/IMAGE');
}

Background Position

Allows you to move a background image (or gradient) around within its container.

It has three different types of values:

  • Length values (e.g. 100px 5px)
  • Percentages (e.g. 100% 5%)
  • Keywords (e.g. top right

header {
    background-position: 100px 5px;
}

Radial Gradients

Creates an image consisting of a progressive transition between two or more color in a radius.

Center


header {
     background-image:  -webkit-radial-gradient(circle cover, red, orange);
}

Top Left


header {
     background-image:  -webkit-radial-gradient(0 0, circle, red, orange);
}

Middle Left


header {
     background-image:  -webkit-radial-gradient(0 50%, circle, red, orange);
}

Linear Gradients

Creates an image consisting of a progressive transition between two or more colors along a straight line.

Top to bottom


			header {
			     background-image:  -webkit-linear-gradient(red, orange);
			}

Right to left


			header {
			     background-image:  -webkit-linear-gradient(right, red, orange);
			}

Using Degrees


			header {
			     background-image:  -webkit-linear-gradient(45deg, red, orange);
			}

RGBA

Controls the opacity of a color

Color property using (red, green, blue, opacity)

opacity is a decimal value from 0 to 1

  • 0 - not visible
  • 1 - fully visible

.example {
background:   rgba(255, 0, 0, 0.8);
}

Opacity

Controls the opacity of an element

opacity is a decimal value from 0 to 1

  • 0 - not visible
  • 1 - fully visible

.example {
opacity:   0.8;
}

Background size

Allows you to resize the background inside its container.

Cover

Tells the browser to make sure the image always covers the entire container, even if it has to stretch the image or cut a little bit off one of the edges. 


header {
    background-size: cover;
}

Contain

Tells the browser to always show the whole image, even if that leaves a little space to the sides or bottom.


header {
    background-size: contain;
}

One Value

If you only provide one value (e.g. background-size: 400px) it counts for the width, and the height is set to auto. You can use any CSS size units you like, including pixels, percentages, ems, viewport units, etc.


header {
    background-size: 400px;
}

Two Values

If you provide two values, the first sets the background image's width and the second sets the height. Like the single value syntax, you can use whatever measurement units you like.


header {
    background-size: 400px 100px;
}

Text-shadow

Add a drop shadow to a text element to make it stand out against its background.

  • The first value is the X-coordinate
  • Second value is the Y-coordinate
  • Third value is the blur radius
  • Fourth value is the color of the shadow

h1 {
 text-shadow:  2px 2px 10px red;
}

Box-shadow

Adds a drop shadow to an element

  • The first value is the offset-x
  • Second value is the offset-y
  • Third value is the blur
  • Fourth value is the spread
  • Fifth value is the color of the shadow

h1 {
 box-shadow:   0 0 5px 5px red;
}

Border Radius

Give any element "rounded corners" by applying a border-radius through CSS.

You'll only notice if there is a color change involved. For instance, if the element has a background-color or border that is different than the element it's over. 


.example {
    border-radius:  10px;
    background:   #BADA55;
}

Two Values

10px radius on top left & bottom right

40px on top right & bottom left


.example {
   border-radius: 10px 40px;
   background:   #BADA55;
}

Four Values

10px radius on top left

25px radius top right

50px radius bottom right

100px radius bottom left


.example {
   border-radius: 10px 40px;
   background:   #BADA55;
}

Make Circles from Squares

50% radius on all corners of a container or image that is a square


.example {
   border-radius: 50%;
   background:   #BADA55;
}
