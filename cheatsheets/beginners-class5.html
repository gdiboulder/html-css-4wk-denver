Fixed Container

The container will be 1440px wide and centered on the screen if the viewport (browser window) is larger than 1440px.


.container {
   width: 1440px;
	 margin: 0 auto;
}

Fluid Container

The container will be 80% of the viewport (browser window) and centered within it.


.container {
   width: 80%;
	 margin: 0 auto;
}

Position: Fixed

Fixed positioning is similar to an aboslute positioned element in that it is taken out of flow. However, it is positioned relative to the viewport - no matter what container it is in.


selector {
   position: fixed;
   top: 0;
   left: 0;
}

Google Fonts

Google has hundreds of free, open-source fonts that have been optimized for the web, and ready for us to use!

The service runs on Google's servers which are fast, reliable and tested. Google provides this service free of charge.

Visit: www.google.com/fonts

Using Google Fonts

  1. Search the hundreds of font families, when you've found a font you like, click the "Select This Font" button on the font page.
  2. Compare and refine the collection.
  3. Copy the import code and place it in the top in your styles css file.

@import url(http://fonts.googleapis.com/css?family=Lato:300,400,700,300italic,400italic|Bree+Serif);

Style text on your site


body {
    font-family: 'Lato', sans-serif;
}

h1, h2, h3, h4, h5, h6 {
    font-family: 'Bree Serif', serif;
}
